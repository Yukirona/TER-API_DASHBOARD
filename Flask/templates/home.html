Bienvenue sur le dashboard, ceci est une page d'explication afin de mieux comprendre le fonctionnement du site.

Tout d'abord vous etes sur la page d'acceuil quand vous recherger le site cest ici que vous arriverez, vous avez le menu disponible sur la gauche que vous pouvez rétracter avec le bouton associé.
Ce menu contient les menus déroulants corrspondants aux catégories de données surt la collectivitée. Ces menus déroulants contiennent les différents liens vers les tableaux correspondant a la nomenclature du site de l'Insee, l'ors d'un clic sur l'un des lien le tableau va etre généré ainsi que la possibilite d'afficher un graphique si disponible avec l'appartion d'un bouton chart en haut de page.

lorsque vous cliquer sur ce bouton pour générer un graphique il apparait en dessous du tableau, un menu en haut a gauche du tableau vous propose d'extraire les données sous differents formats pdf , svg ect.
